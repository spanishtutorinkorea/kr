## How to Deploy for Free (5 minutes)

1. Go to https://github.com/new
2. Repository name: `spanishtutorinkorea` (or any name)
3. Make it **Public**
4. Click "Create repository"

5. Upload these files (index.html + README.md) to the repo

6. Go to **Settings** → **Pages** → Source: Deploy from branch → main → Save

Your site will be live at: https://spanishtutorinkorea.github.io

(Replace with your repo name)
